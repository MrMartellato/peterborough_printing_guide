# Images Directory

Place the following images in this directory:

1. **windows key.png** and **search bar.png** - Windows search interface showing "STUDENT PRINTER.bat" search results

2. **PB_Xerox_BW.png** - Print dialog showing the destination selection with PB_Xerox_B&W option

3. **PB_Xerox_Colour.png** - "Select a destination" dialog showing printer options (PB_Xerox_B&W, PB_Xerox_Colour, etc.)

4. **select document.jpg** - PaperCut MF printer touchscreen interface showing print jobs

All images should be in PNG or JPG format and will be automatically resized to fit the page layout.
